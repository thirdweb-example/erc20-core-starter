# ERC20 Core Starter

Starter template to build an ERC-20 Core Contract using the Modular Contract framework

## Features

- Modular Contract framework
- Forge pre-installed
- Solady pre-installed

## Run locally

Compile contract 

```
forge build
```

## Deploy Contract

To run this contract, run the following in the CLI

```
npx thirdweb deploy
```
